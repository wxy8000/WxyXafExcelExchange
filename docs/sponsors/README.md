# 赞助二维码 / Sponsor QR Codes

此目录用于存放项目的收款二维码图片。

## 需要的文件

请将以下两张二维码图片保存到此目录:

1. `wechat-pay.jpg` - 微信支付收款码
2. `alipay.jpg` - 支付宝收款码

## 说明

- 图片建议尺寸: 300x300 像素
- 格式: JPG 或 PNG
- 文件名必须严格按照上述命名

## 更新引用

添加图片后,以下文件会自动引用这些图片:

- `SPONSORS.md`
- `README_NUGET.md`
- `.github/FUNDING.yml`
- `README.md`

## 二维码获取

如果您需要重新生成二维码,可以从原始URL获取:

- 微信支付: (您提供的微信收款码)
- 支付宝: (您提供的支付宝收款码)
